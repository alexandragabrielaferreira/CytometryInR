new_dataset = col_renamed_dataset |> select("bid", "timepoint", "Condition", "Monocytes", "CD45_count") |> filter(!Condition %in% "PPD", timepoint %in% "9")

Problem 3

“We can also use mutate() to combine columns. For our dataset, “bid”, “timepoint”, “Condition” are separate columns that originally were all part of the filename for the individual .fcs file. Try to figure out a way to combine them back together using paste0(), and save the new column as “filename”. Once this is done, pull() the contents of this column, and using try to determine whether there were any duplicates (think innovative ways of using !, length() and unique())”

Solution

# get specimen names 1st from bid column

specimen_name= working_dataset |> pull("bid")
specimen_name


# get timepoint portion from timepoint column:

specimen_timepoint = working_dataset |> pull("timepoint")
specimen_timepoint


# get condition from condition column:

specimen_condition = working_dataset |> pull("Condition")
specimen_condition


# Create file names from the previous three variables:
original_name = paste0(specimen_name, "_", specimen_condition, "_", specimen_timepoint)
original_name

# Add original file names to the working dataset:

add_filenames = working_dataset |> mutate(Filename = original_name) |> relocate ("Filename", .before = "bid")
head(add_filenames, 6)
glimpse(add_filenames)

# check if there are duplicates:

length(unique(original_name)) == length (working_dataset |> pull("bid"))

I first pulled the info from “bid”, “condition” and “timepoints”:

Second, I created the file names using paste0():

After, I added the file names as a new column and relocated it to the first column of the dataset:

Finally, by checking if the length of the list with unique original_names is equal to length of the number of rows of any column present in the original data set I can see if there are any duplicate files. If there were any duplicates inside the original dataset, the list of unique original_names would be shorter than the number of rows inside the dataset and the result of the logical question would output FALSE.
